# cuchillo-de-gaucho 
 
A collection of useful utility functions for everyday use. 
 
## Installation 
You can install cuchillo-de-gaucho via pip: 
 
``` 
pip install cuchillo-de-gaucho 
``` 
 
## Usage 
Example of usage: 
 
```python 
from cuchillo_de_gaucho import add, multiply 
print(add(2, 3)) 
print(multiply(2, 3)) 
```
